Project Title:

Demanding job Analysis- A data-driven project that identifies the most in-demand and high-paying job roles based on randomly collected LinkedIn profile data. This analysis helps job seekers understand which skills, certifications, and roles are trending in the market.

Description:

This project analyzes a collection of random LinkedIn profiles and generates insights about job demand and salary trends.
Additional columns were created for skills, certifications, demand score, salary score, ranking, and other calculated metrics.
Using data analytics techniques, the project ranks the most demanding jobs and the highest-paid roles based on skills, income, and certification strength.
The goal of this project is to help job seekers and learners understand which profiles and skills lead to better career opportunities.
